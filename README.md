# Aventuras en la Marca del Este

Implementación del sistema de juego Aventuras en la Marca del Este para Foundry Virtual Tabletop.

## Instalación

Puedes descargar el último archivo en github o usar el enlace al manifiesto:
https://raw.githubusercontent.com/Were666/ame/main/system.json
